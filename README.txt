MessageMerge
By: Hendy Irawan <hendy@soluvas.com>

MessageMerge allows the user to send personalized messages to chosen contacts from the Phone Book, based on a customized message template.

User can write messages using flexible template variables. Templates can be loaded, edited, and saved for reuse.

MessageMerge is fully-functional on Symbian platform and emulator, and also tested on Windows and Linux.
